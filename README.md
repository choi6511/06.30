# Simple Portfolio Website

This is a simple responsive portfolio website built with HTML, CSS, and basic JavaScript.

## Pages

- `index.html` - Self-introduction page
- `activities.html` - Activity list page

## Files

- `index.html`
- `activities.html`
- `style.css`
- `script.js`

## How to Run

Open `index.html` in a web browser.

## GitHub Pages

After pushing this project to GitHub:

1. Go to the repository settings.
2. Open the Pages menu.
3. Select the `main` branch.
4. Save the setting.
5. Open the published GitHub Pages URL.
